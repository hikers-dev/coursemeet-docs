# CourseMeet

> 지역 운동 코스 데이터를 시각화해 로컬 상권과 연계한 지역 특화 운동 플랫폼

## ✍️ 서비스 소개

CouseMeet은 사용자의 실제 운동 경로를 기록하고  
이를 ‘코스’ 단위로 공유할 수 있는 위치 기반 운동 서비스입니다.
